Примитивы:
- при let срабатывает присвоение типа
let num = 123 //number
let str = '123' //string

- при const присваивается литеральное значение, вместо типа
const num = 123 //присваивается  123 вместо типа number
const str = '123' //присваивается '123' вместо типа string

- в классах тоже самое, что и с const
Class Num {
readonly num = 123 //присваивается 123 вместо number
}

Never
Функция вернет Never в двух случаях:
- когда бесконечный цикл
- и когда new throw Error()
